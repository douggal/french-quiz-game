This project is provided **as-is**, without any warranty or official support. It is intended for educational and experimental purposes only.

## Reporting Security Issues

As this project is not actively maintained, there is no formal process for reporting security vulnerabilities. However, if you discover a security issue, you may open an issue on the project's GitHub repository. Please include as much detail as possible to help understand and reproduce the issue.
